O que é o PlugMobile?

O PlugMobile é uma suite de aplicativos mobile, com interface rápida, bonita e intuitiva, que você pode integrar ao seu software desktop para oferecer ao usuário uma incrível experiência mobile.

Compatível

Não importa a linguagem de programação do seu software: o PlugMobile é compatível com todas!

Fácil

Seu sistema envia informações para o Plug via API, arquivos de texto ou conexão direta com seu banco de dados. Você escolhe.

Atraente

Os aplicativos PlugMobile foram desenvolvidos para serem extremamente rápidos, bonitos e fáceis de usar. Do jeito que o usuário gosta!

Todo seu

Todos os plugs são white-label, isto é, carregam a sua marca. Personalize o nome do app, o ícone, a página inicial, e até a apresentação dele na AppStore e no Google Play.
